---

## 📖 项目概述

### 项目定位

| 属性 | 内容 |
|------|------|
| **名称** | 世纪公园探索年（辰辰的冒险） |
| **周期** | 48周（2026年5月24日 - 2027年5月29日） |
| **地点** | 郑州世纪公园（管城回族区石化路1号） |
| **核心参与人** | 辰辰（儿子，主要）+ 爸爸/妈妈（陪伴引导） |
| **频次** | 每周末一次，每次1-2小时 |

### 核心目标

1. **系统性探索**：用一年时间系统化认识公园的四季变化
2. **能力成长**：培养观察力、记录习惯、科学思维
3. **亲子陪伴**：爸爸/妈妈陪伴引导，不主导，跟着辰辰的节奏
4. **成果沉淀**：48周后形成完整的探索图鉴和冒险纪事

### 照片上传流程

1. 打开首页 → 点击「照片上传」链接

## 🔄 项目成长性设计

> **总则是底层世界观，不是天花板**

### 核心逻辑

- 总则第14章的48周计划 = 游戏基础地图，是起点不是终点
- 每周根据辰辰的反馈（兴趣度、完成度、理解深度），动态调整任务难度和深度
- 辰辰能力涨了，任务就得跟着涨；总则只是兜底，不封顶

### 可调整维度

| 维度 | 调整条件 | 调整方式 |
|------|---------|----------|
| 任务深度 | 辰辰完成得很轻松 | 下周加难度：从「找3种」升级到「对比2种异同」 |
| 任务广度 | 辰辰对某个主题很感兴趣 | 下周扩展：从观察扩展到创作/实验 |
| 天气适配 | 下周天气预报暴雨/极端天气 | 提前改室内替代任务 |
| 节日/临时事件 | 正好赶上端午/暑假/特殊日子 | 临时插入相关主题任务 |
| 难度升级 | 月度回顾发现连续3周轻松完成 | 整体提升下周任务难度基准 |

### 调整幅度规则

- **微调**：每周根据当周反馈调整下一周（允许）
- **中调**：每月回顾时根据整月数据调整下月任务难度（允许）
- **大调**：修改总则本身的48周计划（不允许，除非辰辰能力出现质变）
