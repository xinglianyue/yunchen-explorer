# 🌳 世纪公园探索图鉴

> 辰辰的48周公园冒险之旅 —— 亲子自然探索项目

## 📖 项目简介

这是一个为期48周的亲子公园探索项目，旨在通过每周一次的公园探索活动，培养孩子的观察力、记录习惯和科学思维，同时增进亲子陪伴时光。

## 🎯 核心目标

- **系统性探索**：用一年时间系统化认识公园的四季变化
- **能力成长**：培养观察力、记录习惯、科学思维
- **亲子陪伴**：爸爸/妈妈陪伴引导，跟着孩子的节奏
- **成果沉淀**：48周后形成完整的探索图鉴和冒险纪事

## 📅 项目周期

## 🎮 使用说明

### 照片上传流程

1. 打开首页 → 点击「照片上传」
2. 选择周次（如第2周 - 公园地图）
3. 填写照片描述
4. 点击上传区域或拖拽照片到上传区
5. 点击「保存照片」完成上传

### 每周任务流程

1. **周五**：发布下周任务卡 + 家长手册
2. **周六/周日**：执行公园探索任务
3. **周日晚上**：填写问卷
4. **周一**：回收问卷 → 更新图鉴 → 计算星星 → 检查徽章/称号 → 发回顾
