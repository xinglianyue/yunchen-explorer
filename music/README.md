# 宝可梦四季音乐文件夹

请将对应季节的宝可梦音乐文件放入此文件夹：

## 文件命名规则

- `summer.mp3` - 夏季主题（第1-12周）
- `autumn.mp3` - 秋季主题（第13-24周）
- `winter.mp3` - 冬季主题（第25-36周）
- `spring.mp3` - 春季主题（第37-48周）

## 推荐的宝可梦音乐

- **夏季 (summer.mp3)**: 真新镇、大木研究所、自行车道
- **秋季 (autumn.mp3)**: 金黄市、枯叶港、自行车道
- **冬季 (winter.mp3)**: 紫苑镇、红莲岛、冠军之路
- **春季 (spring.mp3)**: 华蓝市、常磐森林、宝可梦联盟

## 注意事项

- 文件格式：MP3
- 建议文件大小：控制在 5MB 以内
- 建议时长：2-5分钟循环
