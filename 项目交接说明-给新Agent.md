## 一、项目概述

### 项目定位
- **名称**：世纪公园探索年（辰辰的冒险）
- **周期**：48周（2026年5月24日 - 2027年5月29日）
- **地点**：郑州世纪公园（管城回族区石化路1号，家门口对面）
- **核心参与人**：辰辰（儿子，主要）+ 爸爸/妈妈（陪伴引导）
- **频次**：每周末一次，每次1-2小时

### 项目目标
1. **系统性探索**：用一年时间系统化认识公园的四季变化
2. **能力成长**：培养观察力、记录习惯、科学思维
3. **亲子陪伴**：爸爸/妈妈陪伴引导，不主导，跟着辰辰的节奏
4. **成果沉淀**：48周后形成完整的探索图鉴和冒险纪事

## 六、核心规则（必须遵守）

### ⚠️ 权威文件规则
- **总则手册v3是唯一权威**，任何内容与总则冲突的，以总则为准
- 本交接说明只是操作级备忘，不替代总则

### ⚠️ 任务设计规则
- **必须对照总则第十四章的48周计划**，不准自行发挥
- 第2周是"公园地图"，不是"树木侦探"（树木是第5-8周的事）
- 任务卡格式固定：发现任务 → 挑战任务 → 记录任务 → 回家任务 → 给大人的贴士

### ⚠️ 家长手册规则
- 格式固定：核心目标 → 重点关注 → 引导话术 → 建议装备 → 注意事项 → 问卷链接

### ⚠️ 称呼规则
- 文档里统一：爸爸 / 妈妈 / 辰辰（总则原文用"云辰"，文档里统一用"辰辰"）
- 冒险纪事里陪同人写「妈妈」，辰辰说的话写「辰辰说」
- 爸爸和AI之间内部沟通可以写「辛昕」「张冬梅」

### ⚠️ 项目成长性规则
- 总则是底层世界观，不是天花板
- 整个项目要随着辰辰的成长不断升级
- 每周根据辰辰的反馈（兴趣度、完成度、理解深度），动态调整任务难度和深度
- 辰辰能力涨了，任务就得跟着涨；总则只是兜底，不封顶

### 调整幅度规则
- **微调**：每周根据当周反馈调整下一周（允许，总则允许妈妈当场简化）
- **中调**：每月回顾时根据整月数据调整下月任务难度（允许，总则月度回顾条款）
- **大调**：修改总则本身的48周计划（不允许，除非辰辰能力出现质变、需要跨季重设）
